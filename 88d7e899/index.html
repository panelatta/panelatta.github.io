H. Ryuji doesn't want to study (树状数组) [2018 ACM-ICPC Xuzhou Online Contest]

题面

Description

Ryuji is not a good student, and he doesn’t want to study. But there are n books he should learn, each book has its knowledge a[i]a[i]a[i].

Unfortunately, the longer he learns, the fewer he gets.

That means, if he reads books from ll to rr, he will get a[l]×L+a[l+1]×(L−1)+⋯+a[r−1]×2+a[r]a[l] \times L + a[l+1] \times (L-1) + \cdots + a[r-1] \times 2 + a[r]a[l]×L+a[l+1]×(L−1)+⋯+a[r−1]×2+a[r](LLLis the length of [ $ l, r$ ] that equals tor−l+1r - l + 1r−l+1).

Now Ryuji has qqq questions, you should answer him:

  • If the question type is 111, you should answer how much knowledge he will get after he reads books [ l,rl, rl,r ].

  • If the question type is 222, Ryuji will change the iiith book’s knowledge to a new value.

Input

First line contains two integers nnnandqqq (n,q≤100000n, q \le 100000n,q≤100000).

The next line contains n integers represent a[i]a[i]a[i] (a[i]≤1e9)(a[i]≤1e9)(a[i]≤1e9) .

Then in next qqqline each line contains three integers a,b,ca, b, ca,b,c, if a=1a = 1a=1, it means question type is 111, and b,cb, cb,crepresents [ l,rl , rl,r]. if a=2a = 2a=2, it means question type is222 , andb,cb, cb,c means Ryuji changes thebbbth book’s knowledge toccc .

Output

For each question, output one line with one integer represent the answer.

分析

注意到题中公式可进行如下变形：

a[l]×L+a[l+1]×(L−1)+⋯+a[r−1]×2+a[r]a[l] \times L + a[l+1] \times (L-1) + \cdots + a[r-1] \times 2 + a[r]a[l]×L+a[l+1]×(L−1)+⋯+a[r−1]×2+a[r]

  • = ∑k=lr(r−l+1−(k−l))⋅a[k]\sum\limits_{k=l}^{r}(r-l+1-(k-l)) \cdot a[k]k=l∑r​(r−l+1−(k−l))⋅a[k]
  • = ∑k=lr(r−k+1)⋅a[k]\sum\limits_{k=l}^{r}(r-k+1) \cdot a[k]k=l∑r​(r−k+1)⋅a[k]
  • = (r+1)∑k=lra[k]−∑k=lrk⋅a[k](r+1)\sum\limits_{k=l}^{r}a[k]-\sum\limits_{k=l}^{r}k \cdot a[k](r+1)k=l∑r​a[k]−k=l∑r​k⋅a[k]

又，题中所给两种操作分别为区间查询，单点修改；因此用树状数组直接维护 a[k]a[k]a[k]和k⋅a[k]k \cdot a[k]k⋅a[k] 的值即可。
