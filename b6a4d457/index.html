Codeforces 1133D. Zero Quantity Maximization (map, 精度)

题面

Description

You are given two arrays aaaand bbb, each containsnnn integers.

You want to create a new array cccas follows: choose some real (i.e. not necessarily integer) number ddd, and then for every i∈[1,n]i \in [1, n]i∈[1,n] letci:=d⋅ai+bic_i := d \cdot a_i + b_ici​:=d⋅ai​+bi​.

Your goal is to maximize the number of zeroes in array ccc. What is the largest possible answer, if you choose ddd optimally?

Input

The first line contains one integer nnn (1≤n≤2⋅1051 \le n \le 2 \cdot 10^51≤n≤2⋅105) — the number of elements in both arrays.

The second line contains nnnintegers a1a_1a1​, a2a_2a2​, …,ana_nan​ (−109≤ai≤109-10^9 \le a_i \le 10^9−109≤ai​≤109).

The third line contains nnnintegers b1b_1b1​, b2b_2b2​, …,bnb_nbn​ (−109≤bi≤109-10^9 \le b_i \le 10^9−109≤bi​≤109).

Output

Print one integer — the maximum number of zeroes in array ccc, if you choose ddd optimally.
