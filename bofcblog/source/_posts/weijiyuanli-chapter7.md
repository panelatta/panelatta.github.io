## 8253 内部结构与引脚信号

![](https://cdn.bofc.tech/hexo/weijiyuanli/7-1.png)

### 数据总线缓冲器

与数据总线连接，CPU 对 8253 输出指令进行读/写操作的信息均经由 8 位数据总线 D7~D0 传送，包括：

1. 对 8253 进行初始化编程时写入的控制字。
2. 向某一计数器写入计数初值
3. 读取某一计数器的计数值

### 读/写控制电路

 R̅D̅ : 读信号，低电平有效

 W̅R̅ : 写信号，低电平有效

 C̅S̅ : 片选信号，只有低电平时 CPU 才能对 8253 进行读写操作

A0, A1: 端口选择信号，包括

| A0A1 |   选中端口   |
| :--: | :----------: |
|  00  |   计数器 0   |
|  01  |   计数器 1   |
|  10  |   计数器 2   |
|  11  | 控制字寄存器 |

### 控制字寄存器

![](https://cdn.bofc.tech/hexo/weijiyuanli/7-2.png)

1. 通道选择位
   1. 00：计数器 0
   2. 01：计数器 1
   3. 10：计数器 2
2. 读/写操作位
   1. 00：数据寄存器中的值送锁存器，供 CPU 读
   2. 01：只读/写计数器低字节
   3. 10：只读/写计数器高字节
   4. 11：先读/写计数器低字节，再读/写高字节
3. 工作方式选择位
   1. 000：方式0
   2. 001：方式1
   3. x10：方式2
   4. x11：方式3
   5. 100：方式4
   6. 101：方式5
4. 计数方式选择位
   1. 1：BCD码计数
   2. 0：二进制计数

### 计数器

1. CLK0~CLK2：计数器的输入时钟脉冲从这里输入
2. OUT0~OUT2：计数器的输出端
3. GATE0~GATE2：计数器的门控脉冲输入端

## 8253 初始化

### 初始化步骤

对3个计数器的编程没有先后顺序的规定，只要满足以下顺序：

1. 写入控制字
2. 写入计数初值

例：8253 的 3 个计数器端口地址分别为 `3F0H`, `3F2H`, `3F4H`，控制字寄存器的端口地址为 `3F6H`，要求通道 0 工作于方式 3，且计数初值为 n = 1234H，给出初始化程序。

```assembly
IOY0 	EQU 3F0H
IOY1 	EQU 3F2H
IOY2 	EQU 3F4H
CON8253	EQU 3F6H

SSTACK	SEGMENT STACK
	DW 32 DUP(?)
SSTACK	ENDS

CODE SEGMENT
	ASSUME CS:CODE, SS:SSTACK
START:
	PUSH DS
	MOV AX,	0000H
	MOV DS,	AX			;设DS段基址0000H
	
	MOV AL,	00110111B	;控制字00110111B
	MOV DX, CON8253		;DX指向控制字端口
	OUT DX, AL
	
	MOV DX, IOY0		;DX指向计数器0端口
	MOV AL, 34H			;写入低字节
	OUT DX, AL
	MOV AL, 12H			;写入高字节
	OUT DX, AL
	
	POP DS
CODE ENDS
	END START
```
